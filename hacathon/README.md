# MP Tourism Interactive Experience

## Overview

This project is an interactive tourism experience of Madhya Pradesh built using **p5.js**, **JavaScript**, **HTML**, and **CSS**.
The project contains multiple animated scenes inspired by famous tourist locations of Madhya Pradesh.

Users can navigate between different locations through an interactive map and keyboard controls.

---

# Features

## 🛕 Jatashankar Shivling Scene

* Animated cave environment
* Water droplets falling on Shivling
* Ripple effects in water
* Trishul design
* Water drop sound effects
* Cinematic cave atmosphere

---

## 🐅 Satpura Tiger Reserve

* Animated tiger reserve gate
* Scenic forest environment
* Dynamic sky and landscape
* Interactive jungle safari mode

---

## 🌳 Jungle Safari

* Moving tiger and cub animation
* Animated forest trees
* Cinematic jungle lighting
* Smooth walking animation

---

## 🌊 Bhedaghat Waterfall

* Animated waterfall effect
* Marble rock inspired structures
* Water splash and mist effects
* Beautiful landscape scene

---

## 🚣 Boat Ride Scene

* Interactive boat movement
* Arrow key controls
* Animated river waves
* Marble canyon environment

---

# Interactive Controls

| Key        | Action                   |
| ---------- | ------------------------ |
| S          | Open Shivling Scene      |
| T          | Open Tiger Reserve       |
| G          | Open Bhedaghat Waterfall |
| N          | Open Jungle Safari       |
| W          | Start Boat Ride          |
| B          | Return to Main Menu      |
| Arrow Keys | Move Boat                |
